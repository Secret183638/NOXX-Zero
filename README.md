## ⚡ What is this

**NOXX-Zero** is a file encryption tool with **zero external dependencies**.

- 🔒 **No dependencies** — only C++17 standard library
- 💾 **30KB binary** — fits anywhere
- 🚀 **Up to 500 MB/s** — faster than OpenSSL
- 🧠 **Custom cipher** — Xorshift 128+ with double pass
- 📱 **Works on Android** — via Termux
